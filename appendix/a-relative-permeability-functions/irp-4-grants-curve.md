# IRP=4  Grant's curve

$${k_{rl}}\quad  = \quad {\hat S^4}$$

$${k_{rg}}\quad  = \quad {\left( {1\; - \;\hat S} \right)^2}\;\left( {1\; - \;{{\hat S}^2}} \right)$$

where&#x20;

$$\hat S = \frac{{\left( {{S_l}\; - \;{S_{lr}}} \right)}}{{\left( {{S_l}\; - \;{S_{lr}}} \right)}}$$

with _Slr_ = _RP(1)_; _Sgr_ = _RP(2)_

Restrictions: _RP(1)_ + _RP(2)_ < 1
